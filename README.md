# README

- The game provides a premake script to build the project.
        - The output visual solution is provided
- Do not minimise the game
- The game is unstable on Windows. Works fine on linux.

### Controls
- WASD: movement
- ESC: Quit (in main menu) Pause (in game)
- SPACE: Select (in menu) Jump (in game)
- F1: Toggle mouse relative to screen


### Dependencies
- SDL3
- Vulkan (1.3)
- KTX
- Slang
- FMOD
- TinyOBJLoader
- VMA
- GLM
- VOLK
- Freetype

I have tried to include as many of these as possible.
